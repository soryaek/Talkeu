# Talkeu

<h3>How to run in local environment: </h3>
<ul>
  <li>Install packages: <b>npm install</b> </li>
  <li>Start running:  <b>npm start</b></li>
  <li>Open your web browser and then run: <b>http://localhost:3000/ </b></li>
</ul>
